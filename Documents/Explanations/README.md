# Ghi chú tiếng Việt

Thư mục này dùng để lưu các tài liệu giải thích bằng tiếng Việt cho repo `learn-claude-code`.

Quy ước hiện tại:
- Mặc định ghi chú vào `Documents/Explântions/`.
- Ưu tiên một file cho mỗi chủ đề hoặc mỗi file mã nguồn được giải thích.
- Nội dung nên tập trung vào: mục đích, luồng chạy, thành phần chính, và các điểm đáng chú ý khi đọc code.
